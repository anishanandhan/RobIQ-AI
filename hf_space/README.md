# RedrobIQ-AI Candidate Ranker

An interactive Streamlit-based web dashboard to analyze, filter, and rank candidate profiles for the Senior AI Engineer (Founding Team) role.

## Features
* **InteractiveShortlist**: Displays top candidates with highlights, notice periods, and reasoning.
* **Honeypot Filter**: Showcases faked profiles and logical contradictions detected.
* **Data Visualizations**: Explores candidates using scatter plots, locations bar charts, and notice period distributions.
* **Custom Uploads**: Supports uploading custom JSONL or JSON files to run the ranking algorithm in real time.
